About

Hunter Games is a curated browser game shelf.

The site is built for players who want quick HTML5 games without digging through long, unexplained lists. Every visible game is organized by category and paired with short original notes about the goal, controls, and play style.

Home Contact Privacy Terms

What We Publish

Hunter Games includes local projects, open-source games, open-licensed itch.io entries, and externally hosted HTML5 games that can run inside a modern browser. The goal is to make discovery easier by grouping games into sensible categories and describing them in practical language.

How Games Are Reviewed

Games are checked for basic playability, clear controls, and fit with the current library. Entries may be removed or hidden if they stop loading, have misleading titles, create a poor user experience, or are no longer appropriate for the site.

Why Some Games Open In Frames

Some games are hosted by their original publishers or public game hosts. Hunter Games provides discovery, organization, control notes, and a player interface, while the game itself may load from the source named in the game data.
