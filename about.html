About

Hunter Games is a small maintained game site.

The site is intentionally narrow: it focuses on a local project game, a bundled open-source golf build, and written notes that explain how those games work. That focus keeps the pages easier to maintain and gives visitors a clearer reason to return than a generic list of borrowed embeds.

What We Publish

Hunter Games publishes playable browser entries only when the page can explain the game in its own words. A listing should include the goal of the game, basic controls, where the game files come from, and what kind of session a player should expect.

How Games Are Reviewed

Games are checked for basic playability, clear controls, source clarity, and fit with the current library. Entries may be removed or hidden if they stop loading, have misleading titles, create a poor user experience, or cannot be supported with useful original notes.

Why Games Open In A Player

The homepage uses a single browser player so games can be selected, described, and launched consistently. The visible library is intentionally small so the game files, controls, and page notes can be maintained together.

Update Standard

When the site changes, the goal is to add something a player can actually use: a guide, a clearer control note, a working-game fix, or a better explanation of what was changed. Pages should not exist only to target a search phrase or repeat the same paragraph in a different place.
