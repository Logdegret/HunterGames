How the shelf is built

Curated for short sessions, clear controls, and fewer dead ends.

Many free game portals are just long lists. Hunter Games keeps the front page focused on games that can be described clearly, categorized cleanly, and launched from a modern browser. Games that feel broken, misleading, overly repetitive, or unsuitable for the current library are removed from the visible shelf.

Original play notes

Cards summarize what the game asks you to do, the likely session length, and the controls to try first. The descriptions are written for this site instead of being copied from a store listing.

Source transparency

The library mixes local projects, open-source games, open-licensed itch.io entries, and externally hosted HTML5 games. When a game has a known open license, the card shows it.

Maintenance loop

The player includes a broken-game report button, and the visible collection is filtered so questionable or unreliable entries can be held back while they are reviewed.
